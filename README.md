# Work of Tracker

Bu loyiha PHP (versiya 8), HTML, CSS va MySQL yordamida yaratilgan oddiy ishlovchilarni kuzatish tizimi hisoblanadi. Loyiha davomida foydalanuvchilar ishlovchilarning ismi, kelish va ketish vaqtlarini kiritib, ularning ishlagan soatlarini kuzatish imkoniyatiga ega bo‘ladilar.

## Loyiha Haqida
"Work of Tracker" loyihasi ishlovchilarning ish jadvalini tartibga solish va nazorat qilish uchun mo‘ljallangan. Tizim ishlovchilar kiritgan vaqtlarni tekshirib, ularning ishlash soatlari me'yoridan kam yoki ko‘p ekanligini aniqlaydi va natijani jadval shaklida ko‘rsatadi.

## Texnologiyalar
- **PHP (versiya 8)**: Ma'lumotlar bilan ishlash, CRUD operatsiyalarini amalga oshirish va server tomon kodini yozish uchun foydalanildi.
- **HTML va CSS**: Foydalanuvchi interfeysini yaratish va sahifalarni tartibga solish uchun qo‘llanildi.
- **MySQL**: Ma'lumotlar bazasini boshqarish va ishlovchilar ma'lumotlarini saqlash uchun ishlatilgan.

## Loyiha Xususiyatlari
- Foydalanuvchilar ishlovchilarning ismini, kelish va ketish vaqtlarini kiritishlari mumkin.
- Tizim ishlovchilar ishlagan vaqtni hisoblab, uni normaga solishtiradi.
- Ish vaqti normadan kam yoki ko‘p bo‘lsa, foydalanuvchilarga ma'lumot ko‘rsatiladi.
- Barcha ma'lumotlar jadval ko‘rinishida tartiblanib namoyish etiladi.

## O‘rnatish
1. **Kodni yuklab olish**: Loyiha kodini yuklab olish uchun [GitHub Repository nomi] sahifasiga tashrif buyuring va yuklab oling.
2. **PHP va MySQL muhitini sozlash**: PHP (versiya 8) va MySQL o‘rnatilganligiga ishonch hosil qiling.
3. **Ma'lumotlar bazasini sozlash**:
   - MySQL serverida yangi ma'lumotlar bazasini yarating.
   - Loyiha kodidagi ma'lumotlar bazasiga ulanish parametrlarini moslang.
4. **Loyihani ishga tushirish**: Loyiha fayllarini serverga joylashtirib, ishga tushiring.

## Foydalanish
1. Foydalanuvchi interfeysi orqali ishlovchi ma'lumotlarini kiriting.
2. Kelish va ketish vaqtlarini kiritib, "Saqlash" tugmasini bosing.
3. Jadvalda ishlovchilarning ish vaqtlari natijalarini ko‘ring.
